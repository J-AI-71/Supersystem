SafeShare als PWA installieren & offline nutzen

SafeShare funktioniert offline auf Kernseiten. Installation als PWA wird empfohlen.

Schritte

  1. SafeShare öffnen: Startseite oder App.
  2. Im Browser „Zum Home-Bildschirm hinzufügen“ wählen.
  3. Status prüfen: Status → „Registriert: aktiv“.
  4. Offline testen: Flugmodus aktivieren und App öffnen.

Fehlerbehebung

  • „Wartende Version aktivieren“ auf Status.
  • Mit ?nocache=YYYYMMDD neu laden.
