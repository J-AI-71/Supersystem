Hilfe

Diese Seite erklärt Einrichtung und Nutzung von SafeShare in wenigen Schritten.

1) App

App öffnen

  • Link einfügen → Säubern & öffnen.
  • Kopieren schreibt den bereinigten Link in die Zwischenablage (Fallbacks für iOS).
  • Teilen nutzt das native Share-Sheet, sonst Kopier-Fallback.
  • Prüfen zeigt die Analyse in einem neuen Tab.

2) Bookmarklet

Zur Bookmarklet-Seite

iPhone/iPad

  1. Code kopieren tippen.
  2. Safari: neues Lesezeichen anlegen.
  3. Im Feld Adresse/URL den kopierten javascript:-Code einfügen.

Desktop

Bookmarklet-Button in die Lesezeichenleiste ziehen. Der Linktext wird der Name des Lesezeichens.

3) Pro aktivieren

  1. Pro auf Payhip kaufen.
  2. Nach dem Kauf leitet Payhip auf pro-activate.html?pro=1 um.
  3. Pro ist danach lokal im Browser aktiv (localStorage:ss2_pro=1).

Pro-Extras: Bulk-Cleaning und Team-Whitelist/Publisher-Modus.

Bulk-Cleaning öffnen Team-Setup

4) Team-Setup (Publisher/Whitelist)

  • Publisher-Modus: Tracking-Keys, die in der Whitelist stehen, werden beibehalten.
  • Whitelist-Format: Kommagetrennt, z. B. tag,# (Hash behalten, Amazon-tag behalten).
  • Speicherung lokal im Browser: localStorage:ss_mode, localStorage:ss_wl.

5) Fehlerbehebung

  • Alte Inhalte? tools.html öffnen → „Service Worker abmelden“ → „Alle Caches löschen“.
  • iOS-Homescreen-Icon nicht aktualisiert? Homescreen-Shortcut löschen → Safari Website-Daten für j-ai-71.github.io löschen → neu hinzufügen über „Zum Home-Bildschirm“.
  • Bookmarklet zieht nicht? Prüfen, ob die Lesezeichenleiste sichtbar ist; sonst „Code kopieren“ und manuell als URL einfügen.

6) Datenschutz

SafeShare arbeitet 100 % client-seitig. Es gibt keine Server-Logs, keine Cookies, kein Tracking. Siehe Datenschutz.
