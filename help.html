Hilfe

Kurz und kompakt: So verwendest du SafeShare effizient.

Startseite

1) App

  • App öffnen → Link einfügen → „Säubern“.
  • „Säubern & öffnen“ startet die bereinigte Zielseite direkt.
  • „Prüfen“ zeigt, was entfernt würde, ohne zu ändern.

2) Bookmarklets

  • Desktop: Buttons von Bookmarklets in die Lesezeichenleiste ziehen.
  • Mobil: Lesezeichen speichern → URL durch „Nur Code kopieren“ ersetzen (beginnt mit javascript:).

3) Bulk-Clean

  • Viele Links einfügen (Bulk-Clean) → Optionen wählen → „Säubern“.
  • Export als TXT, Duplikate optional entfernen.

4) Team-Whitelist

  • Legitime Parameter (z. B. tag, ref) in Team-Setup hinterlegen.
  • Gilt automatisch für App, Bookmarklets und Bulk-Clean.

5) SafeShare Pro

  • Pro kaufen: Payhip.
  • Nach Kauf: automatische Weiterleitung zu pro-activate.html?pro=1 → Pro wird lokal aktiviert.
  • Zur Kontrolle „Pro aktiv“ in App/Status prüfen.

6) Status & Updates

  • Status → „Auf Updates prüfen“ → bei Bedarf „Wartende Version aktivieren“.
  • Bei Darstellungsfehlern mit ?nocache=YYYYMMDD neu laden.

7) Offline & PWA

  • Als PWA installierbar; nach dem ersten Laden offline nutzbar.
  • iOS: Teilen → „Zum Home-Bildschirm“. Android: Installationsbanner/„Zum Startbildschirm“.

Anleitungen (How-tos)

  • UTM-/Tracking-Parameter entfernen
  • Weiterleitungen entpacken (Klarziel)
  • Bulk-Clean: viele Links auf einmal säubern
  • Bookmarklet installieren & nutzen
  • Team-Whitelist konfigurieren
