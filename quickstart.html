Startseite App Hilfe

So startest du in 60 Sekunden

  1. Öffne die App: app.html.
  2. Füge einen Link ein und klicke auf „Säubern“.
  3. Ergebnis kopieren, teilen oder öffnen.

Optional

  • Bookmarklets: bookmarklets.html → Button in Lesezeichenleiste ziehen (Desktop) oder Code in Lesezeichen einsetzen (Mobil).
  • Team-Whitelist: team-setup.html → legitime Parameter (z. B. tag, ref) hinterlegen.
  • Bulk-Clean für Listen/CSV: bulk-clean.html.
  • Installierbar als PWA. iOS: Teilen → „Zum Home-Bildschirm“. Android/Chrome: „Installieren“.

Drucken: Strg+P / ⌘P. Diese Seite ist für eine DIN-A4-Seite optimiert.

Was SafeShare entfernt

Tracking-Parameter wie utm_*, gclid, fbclid, msclkid und ähnliche. Übliche Redirect-Wrapper (z. B. google.com/url, l.facebook.com/l.php) werden entpackt. Alles lokal im Browser – keine App-Server-Logs.
