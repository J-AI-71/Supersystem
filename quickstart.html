SafeShare · Quickstart

60-Sekunden-Start

So nutzt du SafeShare

  1. URL einfügen: Öffne /Supersystem/app.html und füge deinen Link ein.
  2. Säubern: Klicke Säubern – Tracking-Parameter (z. B. utm_*, gclid) werden entfernt.
  3. Öffnen/Teilen: Ergebnis direkt öffnen, kopieren oder teilen.
  4. Bookmarklet (optional): „Nur-Code“ kopieren, als Lesezeichen speichern, URL des Lesezeichens durch den Code ersetzen.
  5. Pro (optional): Für Bulk-Clean & Team-Whitelist: Kauf → Redirect auf pro-activate.html?pro=1.

Tipp (Mobil/PWA): „Zum Home-Bildschirm“ hinzufügen – die App startet dann wie eine native Anwendung.
