Quickstart

SafeShare in 1 Minute: App, Installation (PWA), Bookmarklet, Pro, Team-Whitelist, Troubleshooting.

Startseite

Schritt-für-Schritt

  1. App öffnen: /app.html. URL einfügen → Säubern oder Säubern & öffnen.
  2. Installieren (PWA):
    • iOS Safari: Teilen → „Zum Home-Bildschirm“
    • Android Chrome: Menü → „App installieren“
    • Desktop: Browser „Installieren“-Icon (falls angeboten)
  3. Bookmarklet (optional): /bookmarklets.html öffnen → Button in Lesezeichenleiste ziehen. Auf Mobilgeräten: Lesezeichen-URL durch den javascript:-Code ersetzen.
  4. Pro (optional): Pro kaufen → Weiterleitung oder manuell: /pro-activate.html?pro=1 → Badge „Pro aktiv“ sichtbar (z. B. in Bulk-Clean).
  5. Team-Whitelist: /team-setup.html öffnen → erlaubte Parameter (z. B. tag,ref) eintragen → speichern.
  6. Troubleshooting: /status.html → „Auf Updates prüfen“ bzw. „Wartende Version aktivieren“. Alternativ Seiten mit ?nocache=YYYYMMDD neu laden.

Kurz erklärt

  • Was SafeShare tut

    Entfernt Tracking-Parameter (UTM, gclid/fbclid/msclkid …) und entpackt gängige Weiterleitungs-Wrapper – lokal im Browser.

  • Was erhalten bleibt

    Über die Team-Whitelist können legitime Parameter (z. B. tag, ref) erlaubt werden.

  • Offline & Datenschutz

    PWA-fähig, funktioniert nach dem ersten Laden offline. Keine Server-Logs, keine Konten.
