SafeShare Bookmarklets

Mit einem Klick Links säubern – direkt aus dem Browser. Desktop: per Drag & Drop in die Lesezeichenleiste ziehen. Mobil (iOS/Android): Code kopieren und in ein neues Lesezeichen als URL einfügen.

1) SafeShare Clean (öffnen)

Leitet die aktuelle Seite an die SafeShare-App weiter, dort wird der Link bereinigt und das Ziel geöffnet.

SafeShare Clean

Hinweis: Klicken auf der Seite kann durch CSP blockiert sein – bitte ziehen oder den Code kopieren.

2) Redirect entpacken

Entpackt typische Weiterleitungen (z. B. t.co, l.facebook.com) und zeigt das Klarziel.

Redirect entpacken

3) SafeShare Clean (nur kopieren)

Übergibt die aktuelle URL an die App und zeigt die bereinigte Version zum Kopieren.

Clean → Kopieren

Installation

  1. Desktop (Chrome/Firefox/Edge): Lesezeichenleiste einblenden → den Button oben in die Leiste ziehen.
  2. Safari (macOS): „Darstellung → Favoritenleiste einblenden“ → Button ziehen.
  3. iPhone/iPad (Safari): Zuerst eine beliebige Seite als Lesezeichen speichern → danach in „Lesezeichen bearbeiten“ die URL durch den oben kopierten Code ersetzen.
  4. Android (Chrome/Firefox): Neues Lesezeichen anlegen → Titel vergeben → URL-Feld durch den Code ersetzen.

Tipp: Benenne das Lesezeichen z. B. „SafeShare Clean“, „Redirect entpacken“ oder „Clean → Kopieren“.
