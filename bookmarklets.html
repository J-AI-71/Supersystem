Bookmarklets

Bookmarklets sind Lesezeichen, die ein kleines Script ausführen. Damit kannst du SafeShare auf Desktop sehr schnell nutzen: aktuelle URL übernehmen → SafeShare öffnen → optional direkt bereinigen.

App öffnen Hilfe Pro ansehen
Desktop: Bookmarklets sind eine Abkürzung für Chrome/Firefox/Edge.
iOS/iPad: Am stabilsten ist ein Share-Sheet Shortcut.
iOS Shortcut ansehen

1) App öffnen & URL nur ausfüllen

Übernimmt die aktuelle Seite in SafeShare (Parameter u). Du klickst danach manuell auf „Bereinigen“.

Drag-to-Bookmarks (Desktop)

Zieh den Button in deine Lesezeichenleiste (Chrome/Firefox/Edge). Das ist am zuverlässigsten.

2) App öffnen & sofort bereinigen (1-Klick)

Wie oben, aber mit run=1. SafeShare übernimmt die URL und startet direkt die Bereinigung.

3) „Einfügen“-Bookmarklet

Fragt dich nach einer URL (einfügen) und öffnet SafeShare direkt mit Auto-Bereinigung.

Kein Clipboard-Zugriff: du fügst manuell ein (Browser-Sicherheit).

Einrichten (Desktop)

  1. Lesezeichen erstellen
  2. Name: z. B. SafeShare / SafeShare 1-Klick
  3. URL/Feld: den Bookmarklet-Code einfügen (muss mit javascript: beginnen)
  4. Auf einer Website klicken → SafeShare öffnet
Tipp: Als eine Zeile einfügen (ohne Umbrüche). Drag-Buttons sind noch zuverlässiger.

Fehlersuche

Es passiert nichts

  • Prüfe: beginnt die Lesezeichen-URL wirklich mit javascript:?
  • Wurde der Code gekürzt/umgeschrieben? (passiert bei manchen Browsern)
  • Teste auf einer „normalen“ Website (nicht PDF-Viewer / nicht interne Browserseiten).

SafeShare öffnet, aber Feld bleibt leer / run=1 startet nicht

  • Dann ist es fast immer App-Version oder Cache.
  • Wenn Direkt-Test nicht geht: Service-Worker/Cache aktualisieren.
