Bookmarklets

Desktop: Link in die Lesezeichenleiste ziehen oder Rechtsklick → „Zu Lesezeichen hinzufügen“. iOS: Code kopieren und als Lesezeichen-URL einsetzen.

Affiliate-safe (Provisionen bleiben)

In Lesezeichenleiste ziehen Affiliate-safe (ziehen)

Öffnet die aktuelle Seite in SafeShare. Tracking weg, Affiliate-Keys bleiben.

Strict (alles weg, auch Affiliates)

In Lesezeichenleiste ziehen Strict (ziehen)

Entfernt Tracking plus Affiliate-Parameter.

Installation Desktop

  1. Lesezeichenleiste einblenden:
    • Chrome: Ctrl+Shift+B / ⌘+Shift+B
    • Safari macOS: Darstellung → „Favoritenleiste einblenden“
    • Firefox: Ansicht → Symbolleisten → „Lesezeichen-Symbolleiste → Immer anzeigen“
  2. Oben „ziehen“-Link mit der Maus in die Leiste ziehen. Alternativ: Rechtsklick → „Zu Lesezeichen hinzufügen“.
  3. Test: Seite mit Tracking öffnen → Lesezeichen anklicken.

Installation iOS (Safari)

  1. Beliebige Seite als Lesezeichen speichern.
  2. In den Lesezeichen auf Bearbeiten → Eintrag wählen.
  3. Den URL-Wert komplett durch den kopierten Code ersetzen (beginnend mit javascript:).
  4. Speichern. Danach auf jeder Seite per Lesezeichen ausführen.

Hinweis: Kein Drag-and-Drop auf iOS. Achte auf gerade Anführungszeichen ' und keine Leerzeichen vor javascript:.

Tastenkürzel & Tipps

  • Es gibt keine offiziellen Hotkeys direkt für Lesezeichen. Workaround: Lesezeichen weit links in die Leiste ziehen, kurzer Name, z. B. „S-Safe“ und „S-Strict“.
  • In Chrome/Firefox kann die Leiste per Ctrl+Shift+B / ⌘+Shift+B schnell ein-/ausgeblendet werden.
  • Problem „öffnet als Suche“: In den Lesezeichen prüfen, dass die URL wirklich mit javascript: beginnt, ohne Zeilenumbruch.
