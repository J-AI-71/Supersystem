iOS Safari – Kurz-Anleitung

  1. Beliebige Seite als Lesezeichen speichern.
  2. Im Lesezeichen auf Bearbeiten → Eintrag wählen.
  3. Den URL-Wert durch den kopierten Code ersetzen.
  4. Speichern. Danach auf jeder Seite per Lesezeichen ausführen.
