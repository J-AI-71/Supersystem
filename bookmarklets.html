Bookmarklets

1) Installieren

Desktop: die Links in die Lesezeichenleiste ziehen.

iOS/Android: Lesezeichen anlegen, dann die URL durch den Code ersetzen.

SAFE (Affiliates behalten)

SafeShare SAFE

Code anzeigen 
javascript:(()=>{const u=location.href;
const t='https://j-ai-71.github.io/Supersystem/app.html?aff=1&ao=1&u='+encodeURIComponent(u);
window.open(t,'_blank');})()

STRICT (alles entfernen)

SafeShare STRICT

Code anzeigen 
javascript:(()=>{const u=location.href;
const t='https://j-ai-71.github.io/Supersystem/app.html?aff=0&ao=1&u='+encodeURIComponent(u);
window.open(t,'_blank');})()

2) Klick-Test (öffnet die App mit einer Beispiel-URL)

Hinweis: Auf dieser Seite selbst würden Bookmarklets immer wieder diese Seite öffnen. Zum echten Test eine externe Seite aufrufen und dann das Lesezeichen klicken.

Hinweise

  • Pop-up-Blocker kann neue Tabs verhindern. Klick-Test oben nutzt Navigation im selben Tab.
  • Kurzlinks wie pin.it/… leiten serverseitig. Ergebnis erst nach Laden sichtbar.
