SafeShare Bookmarklets

Bookmarklets starten SafeShare direkt aus deiner Lesezeichenleiste. So kannst du Links auf der aktuellen Seite mit einem Klick säubern.

Installation im Desktop-Browser

  1. Ziehe die folgenden Buttons mit der Maus in deine Lesezeichen-/Favoritenleiste:
SafeShare – Link säubern SafeShare – nur prüfen
  1. Öffne eine Seite mit einem getrackten Link.
  2. Klicke auf das Bookmarklet – SafeShare übernimmt die Reinigung.

Hinweis: Ersetze javascript:void(0) durch deinen echten Bookmarklet-Code, z. B. eine javascript:-URL, die SafeShare im neuen Tab mit der aktiven URL öffnet.

Installation auf Mobilgeräten

Auf iOS/Android kannst du Bookmarklets über „Zu Lesezeichen hinzufügen“ einrichten und den Code später im Lesezeichen bearbeiten:

  1. Diese Seite als Lesezeichen speichern.
  2. Im Lesezeichen die URL durch deine javascript:…-Bookmarklet-URL ersetzen.
  3. Wenn du später einen Link säubern willst: Lesezeichen aufrufen.
