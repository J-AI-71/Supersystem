Bookmarklets

1) Ziehen

Desktop: Link in die Lesezeichenleiste ziehen.

iOS/Android: Lesezeichen anlegen, danach die URL durch den Code ersetzen.

SAFE (Affiliates behalten)

SafeShare SAFE

Code anzeigen 
javascript:(()=>{const u=location.href;
const t='https://j-ai-71.github.io/Supersystem/app.html?aff=1&ao=1&u='+encodeURIComponent(u);
window.open(t,'_blank');})()

STRICT (alles entfernen)

SafeShare STRICT

Code anzeigen 
javascript:(()=>{const u=location.href;
const t='https://j-ai-71.github.io/Supersystem/app.html?aff=0&ao=1&u='+encodeURIComponent(u);
window.open(t,'_blank');})()

Hinweise

  • Pop-up-Blocker kann den neuen Tab verhindern. Dann „gleich“-Links in Tests nutzen.
  • Kurzlinks wie pin.it/… leiten serverseitig. Ergebnis erst nach Laden sichtbar.
