Bookmarklets

SafeShare Bookmarklets – Tracking direkt in der URL entfernen

Bookmarklets sind „Lesezeichen mit JavaScript“: Du speicherst sie wie ein normales Bookmark und kannst dann mit einem Klick Tracking-Parameter aus der aktuellen URL entfernen – ohne Extension, ohne Login.

Zur SafeShare Web-App Mehr Möglichkeiten mit SafeShare Pro

Alles passiert lokal in deinem Browser. Die Bookmarklets schicken keine Daten an SafeShare-Server, sondern arbeiten nur mit der aktuellen Adresszeile.

1. Bookmarklets hinzufügen

Desktop-Browser (Chrome, Firefox, Edge, Safari …)

  1. Ziehe den jeweiligen Button aus der Liste unten in deine Lesezeichen-Leiste.
  2. Oder: Rechtsklick → „Link zu Lesezeichen hinzufügen“.
  3. Dann eine Seite mit Tracking-Parametern öffnen und das Bookmarklet anklicken.

iPad / iPhone (Safari)

  1. Beliebige Seite als Lesezeichen speichern.
  2. In Safari die Lesezeichen öffnen, das neue Lesezeichen bearbeiten.
  3. Den Namen anpassen (z. B. „SafeShare Clean“).
  4. Die URL des Lesezeichens durch den Code aus dem jeweiligen Codeblock unten ersetzen (komplett markieren, einfügen).
  5. Speichern. Beim nächsten Aufruf auf einer Seite mit Tracking-URL ausführen.

Ja, das ist auf iOS etwas umständlicher – dafür laufen die Bookmarklets komplett ohne zusätzliche App oder Extension.

2. SafeShare Bookmarklets

SafeShare: Tracking entfernen & Seite neu laden
SafeShare Clean & Reload

Entfernt bekannte Tracking-Parameter aus der aktuellen URL und lädt die Seite mit der bereinigten Adresse neu. Wenn nichts zu entfernen ist, bekommt du einen kurzen Hinweis.

Code anzeigen (für manuelles Einfügen) 
javascript:(function(){try{var u=new URL(location.href);var del=['utm_source','utm_medium','utm_campaign','utm_term','utm_content','utm_id','utm_name','utm_reader','utm_viz_id','gclid','fbclid','dclid','yclid','twclid','mc_cid','mc_eid','vero_conv','vero_id','igshid','rb_clickid','ml_subscriber','ml_subscriber_hash'];del.forEach(function(p){u.searchParams.delete(p);});var changed=String(u.href)!==String(location.href);if(changed){location.href=u.toString();}else{alert('SafeShare: Keine typischen Tracking-Parameter in der URL gefunden.');}}catch(e){alert('SafeShare: Konnte die URL nicht bereinigen.');}})();
SafeShare: bereinigten Link kopieren
SafeShare Clean & Copy

Entfernt die gleichen Tracking-Parameter wie oben, verändert die aktuelle Seite aber nicht, sondern kopiert die bereinigte URL in deine Zwischenablage (oder zeigt sie in einem Prompt an).

Code anzeigen 
javascript:(function(){try{var u=new URL(location.href);var del=['utm_source','utm_medium','utm_campaign','utm_term','utm_content','utm_id','utm_name','utm_reader','utm_viz_id','gclid','fbclid','dclid','yclid','twclid','mc_cid','mc_eid','vero_conv','vero_id','igshid','rb_clickid','ml_subscriber','ml_subscriber_hash'];del.forEach(function(p){u.searchParams.delete(p);});var clean=u.toString();if(navigator.clipboard&&navigator.clipboard.writeText){navigator.clipboard.writeText(clean).then(function(){alert('SafeShare: Bereinigter Link wurde in die Zwischenablage kopiert.');}).catch(function(){prompt('Bereinigter Link (bitte manuell kopieren):',clean);});}else{prompt('Bereinigter Link (bitte manuell kopieren):',clean);}}catch(e){alert('SafeShare: Konnte die URL nicht bereinigen.');}})();
SafeShare: aktuellen Link in der Web-App öffnen
SafeShare in App öffnen

Öffnet die SafeShare Web-App in einem neuen Tab und übergibt die aktuelle URL als Parameter (?url=…). Wenn deine App den Parameter auswertet, kann sie das Feld automatisch vorausfüllen – sonst kannst du den Link dort manuell weiterverwenden.

Code anzeigen 
javascript:(function(){try{var u=encodeURIComponent(location.href);var base='https://j-ai-71.github.io/Supersystem/app.html';window.open(base+'?url='+u,'_blank','noopener');}catch(e){alert('SafeShare: Konnte die URL nicht übergeben.');}})();

3. Zusammen mit der App nutzen

Die Bookmarklets sind bewusst „arm an Infrastruktur“ und laufen rein im Browser. Wenn du mehr Kontrolle brauchst (z. B. eigene Listen von Parametern, Redirect-Entschachtelung, Vergleiche), nutze zusätzlich:

  • SafeShare Web-App für komplexere Fälle
  • Weiterleitungen entschachteln
  • Link-Health Check

So kannst du spontan im Alltag „Klick & sauber“ machen – und bei Bedarf trotzdem tiefer einsteigen.
