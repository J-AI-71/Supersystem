Zweck der Anwendung

SafeShare bereinigt URLs clientseitig im Browser. Es entfernt Tracking-Parameter und kann Affiliate-Parameter optional behalten.

Verarbeitete Daten

  • Die von dir eingefügten URLs werden nur lokal verarbeitet.
  • Es werden keine Daten an einen Server von SafeShare gesendet.
  • Profile pro Domain werden lokal im localStorage gespeichert und bleiben auf deinem Gerät.

Cookies, Tracking, Analytics

SafeShare setzt keine Cookies und nutzt keine Analytics-Dienste.
