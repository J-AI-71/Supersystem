2. Zweck der Seite

SafeShare bereinigt übergebene URLs um gängige Tracking-Parameter (z. B. utm_*, gclid, fbclid), ohne Nutzertracking.

3. Verarbeitete Daten

  • Aufrufdaten: Beim Aufruf werden durch den Hosting-Anbieter (GitHub Pages) serverseitig Protokolle erzeugt (z. B. IP, Datum/Uhrzeit, angeforderte Datei, User-Agent). Diese Logdaten werden von GitHub verarbeitet.
  • Eingegebene Links: Eingegebene/übergebene URLs werden nur lokal im Browser verarbeitet und bereinigt. Es findet keine Speicherung auf unserem Server statt.
  • Local Storage: Für Pro-Freischaltung wird ein lokales Flag (ss2_pro) im Browser gespeichert.

4. Dritte/Einbindungen
