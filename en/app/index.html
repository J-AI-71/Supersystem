Clean a link

Tracking parameters are extra data on a link — like labels. SafeShare removes common tracking keys locally in your browser. Nothing is uploaded.

Note: This is not an “invisible” promise. It’s link hygiene before sharing.

Paste a URL

Paste → Clean → Copy/Share the result.

Tip: Drop every link here → share clean.

Works well with WhatsApp, Signal, email, Slack — just paste after copying.

Need more control?

See what was removed, keep selected parameters, use policies/audit → SafeShare Pro

Make it a habit

Add a bookmark or install to your home screen so you can clean in one tap next time.

If the cleaned link doesn’t work: open the original → compare. If only the cleaned link fails, a removed parameter was likely functional. Then clean more conservatively (or Pro: audit/allowlist/policies).

What gets removed?

  • utm_* (campaign parameters)
  • Click IDs like gclid, fbclid, msclkid, igshid
  • more common tracking keys (e.g. _ga, mc_cid, mc_eid, srsltid)
Don’t blindly remove “everything”: affiliate/ref keys (ref, tag, …) can be intentional. Pro is for policies + audit + allowlists.
