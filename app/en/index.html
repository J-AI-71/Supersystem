Clean a link

Tracking parameters are extra “stickers” attached to URLs. SafeShare removes common tracking keys locally in your browser. Nothing is uploaded.

Note: this is not an “invisibility” promise. It’s link hygiene before sharing.

Paste a URL

Paste → Clean → Copy/Share.

Result

Open
If the cleaned link breaks: open the original → compare. If only the cleaned link fails, a removed parameter was likely functional. Then clean more conservatively (or use Pro for audit/allowlists).

What gets removed?

  • utm_* (campaign parameters)
  • click IDs like gclid, fbclid, msclkid, igshid
  • other common tracking keys (e.g. _ga, mc_cid, mc_eid, srsltid)
Don’t remove everything blindly: affiliate/partner keys (ref, tag, …) can be intentional. Pro is for policies + audit + allowlists.
