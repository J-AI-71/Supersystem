Link bereinigen

Tracking-Parameter sind Zusatzinfos am Link – wie Etiketten. SafeShare entfernt typische Tracking-Keys direkt im Browser. Nichts wird hochgeladen.

Hinweis: Das ist kein „Unsichtbar“-Versprechen. Es ist Link-Hygiene beim Teilen.

URL einfügen

Link einfügen → Bereinigen → Ergebnis kopieren/teilen.

Ergebnis

Öffnen
Wenn ein Link nach der Bereinigung nicht funktioniert: Original öffnen → bereinigten Link öffnen. Wenn nur der bereinigte Link scheitert, war ein Parameter vermutlich funktional → dann konservativer vorgehen oder Pro (Audit/Allowlist).

Zuletzt: —

Was wird entfernt?

  • utm_* (Kampagnen-Parameter)
  • Klick-IDs wie gclid, fbclid, msclkid, igshid
  • weitere typische Tracking-Keys (z. B. _ga, mc_cid, mc_eid, srsltid)
Nicht blind „alles weg“: Affiliate/Partner-Keys (ref, tag, …) können Zuordnung/Einnahmen steuern. Aggressiv ist Pro (Policies + Audit + Allowlist).
