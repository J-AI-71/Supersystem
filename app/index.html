Link bereinigen

Tracking-Parameter sind Zusatzinfos am Link – wie Etiketten. SafeShare entfernt typische Tracking-Keys lokal im Browser. Nichts wird hochgeladen.

Hinweis: Das ist kein „Unsichtbar“-Versprechen. Es ist Link-Hygiene beim Teilen.

URL einfügen

Einfügen → Bereinigen → Ergebnis kopieren/teilen.

Ergebnis

Öffnen

Tipp: Jeden Link kurz hier rein → sauber teilen.

Funktioniert gut mit WhatsApp, Signal, Mail, Slack – einfach nach dem Kopieren einfügen.

Mehr Kontrolle?

Sieh genau, was entfernt wurde, behalte ausgewählte Parameter, nutze Policies/Audit → SafeShare Pro

Mach’s zur Gewohnheit

Setze ein Lesezeichen oder füge SafeShare zum Home-Screen hinzu, damit du beim nächsten Mal 1-Tap reinigen kannst.

Wenn der bereinigte Link nicht funktioniert: Original öffnen → vergleichen. Wenn nur der bereinigte Link scheitert, war ein entfernter Parameter vermutlich funktional. Dann konservativer reinigen (oder Pro: Audit/Allowlist/Policies).

Was wird entfernt?

  • utm_* (Kampagnen-Parameter)
  • Klick-IDs wie gclid, fbclid, msclkid, igshid
  • weitere typische Tracking-Keys (z. B. _ga, mc_cid, mc_eid, srsltid)
Nicht blind „alles weg“: Affiliate/Partner-Keys (ref, tag, …) können bewusst gesetzt sein. Pro ist für Policies + Audit + Allowlists.
