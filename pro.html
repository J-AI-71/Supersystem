SafeShare Pro aktiv

Pro schaltet Bulk-Cleaning und Team-Whitelist/Publisher-Modus frei. Aktivierung erfolgt lokal im Browser nach dem Kauf.

Funktionen

  • Bulk-Cleaning: viele Links auf einmal bereinigen (bulk-clean.html).
  • Team-Whitelist: erlaubte Parameter definieren (team-setup.html).
  • Publisher-Modus: bestimmte Keys (z. B. Affiliate-Refs) gezielt erhalten.
  • Lokal & offline: keine Accounts, keine Server-Logs, kein Tracking.

So aktivierst du Pro

  1. Kaufen auf der Produktseite.
  2. Nach dem Kauf leitet Payhip auf pro-activate.html?pro=1 um.
  3. Zurück zur Startseite – der Badge zeigt „aktiv“.
Jetzt kaufen Bereits gekauft? Aktivieren Tools öffnen

Hinweis: Pro-Status (ss2_pro=1) wird ausschließlich lokal im Browser gespeichert.

Team-Setup

Definiere, welche Parameter beim Säubern erhalten bleiben (Whitelist). Beispiel:

utm_source, utm_medium, utm_campaign, ref

FAQ (kurz)

Wo wird „Pro“ gespeichert?

Nur lokal in deinem Browser (localStorage ss2_pro). Es gibt keine Server-Accounts, keine Nachverfolgung.

Neuer Browser/Neues Gerät?

Pro erneut aktivieren: pro-activate.html?pro=1 aufrufen (z. B. über After-Purchase-Redirect oder Link oben).

Pro deaktivieren?

tools.html öffnen → „Pro deaktivieren“ oder localStorage leeren.
